저장소 이름 : Power
원격 저장소에 존재하는 브랜치 : main, Develop
git checkout -b 새로운 브랜치 이름 -> 새로운 브랜치 이름으로 브랜치를 만들어
체크아웃하겠다.
checkout : 다른 브랜치로 이동하여라
-b 새로 생성
push를 해주어야 원격리포지토리에 브랜치가 생김
git branch : 로컬 브랜치를 봄
git branch -r 원격 브랜치를 봄
git branch --set-upstream-to=origin/Develop
깃허브에 내가 지금 원격브랜치랑 로컬브랜치랑 같은걸 알려주는 커맨드
업스트림을 원격브랜치랑 일치시켜주면 됨 (이러면 바로 푸쉬가 가능함)
git pull 로컬 브랜치에 원격저장소의 정보를 가져와 머지하겠다.(동기화 하겠다)

v0.0.1 버전에서 맨 왼쪽은 메이져 중간은 마이너 버젼
git fetch -p 새로고침, 패치내역 확인
git branch -D Alpha : 로컬 알파를 지움
git push origin --delete Alpha : 원격 알파를 지움
git merge origin/Develop : 디벨롭에서 병합함!
