# 용어 정리

 - transform.eulerAngles : 트랜스폼의 오일러 각의 값을 설정하거나 받는 프로퍼티

 - Mathf.Deg2Rad : 라디안 값을 각도 값으로 변경해준다. (상수)





 - 오일러각(Euler Angle) : Vector3의 형태로 이루어진 값

 - 복소수 : 실수와 허수를 합친 수

 - 라디안 : '라디안' 이라는 단위로 각도를 표현한 것
    반지름의 길이가 r인 원에서 호의 길이가 r과 같을 때의 각도를 1 라디안 이라고 한다.
    360도 = 2πr 이다.
